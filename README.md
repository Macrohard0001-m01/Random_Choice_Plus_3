## 这玩意儿有啥用？

就是老师上课随机点学生回答问题用的，比传统的"点兵点将"公平一点。


## 主要功能

### 基础功能
- ✅ 从名单里随机抽人
- ✅ 单次不重复点名（直到重置）
- ✅ 简单的历史记录
- ✅ 全屏/窗口切换

### 特色功能  
- 🎯 **爆率系统**：可以手动调某些学生的中奖概率
- 📊 **平衡算法**：尽量让每个人都被点到
- 😴 **休眠模式**：长时间不用会自动休眠
- 🪟 **悬浮按钮**：最小化后有个小按钮可以快速恢复

### 实际体验
- 界面挺简洁的，不花里胡哨
- 运行还算流畅，不卡顿
- 配置文件都是文本，可以自己改（编辑器与源码放一块了，自己找~）



## 怎么用？

### 简单（bushi）用法（推荐）
下载 release 里的版本，按照教程安装开发环境（我摆烂了，不打包了）。

### 开发者用法
如果你要改代码：
```bash
pip install pygame pywin32 PyQt5 chardet
python plus_main.pyw
```


## 文件说明

- `plus_main.pyw` - 主程序
- `name.txt` - 学生名单
- `config.ini` - 配置文件
- `images/` - 图片资源



## 技术栈

- **PyGame** - 主要界面
- **PyQt5** - 悬浮按钮
- **pywin32** - Windows特有功能
